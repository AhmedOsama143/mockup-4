## DESIGN REQUIREMENTS

### NEW & UNIQUE DESIGN DIRECTION

**CRITICAL:** This design must be **completely different** from any previous versions. Consider one of these distinct aesthetic directions (choose ONE and commit fully):

**Option A: Modern Minimalist Elegance**
- Ultra-clean layouts with generous white space
- Thin, elegant typography (serif for headings, sans-serif for body)
- Muted, sophisticated color palette (beiges, soft grays, deep forest green)
- Subtle animations and transitions
- Card-based layouts with soft shadows
- Minimalist icons using pure CSS

**Option B: Bold Contemporary Luxury**
- Strong, geometric layouts with asymmetric elements
- Bold typography with high contrast
- Rich, dramatic color palette (deep emerald, gold accents, charcoal)
- Oversized images and full-bleed sections
- Angular shapes and diagonal lines
- Dramatic hover effects and animations

**Option C: Warm Mediterranean / Desert Aesthetic**
- Organic, flowing layouts
- Warm, earthy color palette (terracotta, sand, olive green, cream)
- Rounded corners and soft edges throughout
- Textured backgrounds (subtle patterns via CSS)
- Hand-crafted feel with elegant curves
- Warm, inviting atmosphere

**Option D: Sleek Scandinavian Modern**
- Grid-based, structured layouts
- Clean sans-serif typography throughout
- Cool, neutral palette (whites, light grays, dusty blue, warm wood tones)
- Functional minimalism with perfect spacing
- Flat design with depth created through layering
- Calm, serene atmosphere

**Option E: Boutique Hotel Luxury**
- Editorial, magazine-style layouts
- Mix of serif and sans-serif fonts
- Sophisticated palette (navy, gold, cream, burgundy)
- Elegant spacing and composition
- Image-first design with overlapping elements
- Refined, upscale atmosphere

**Choose ONE direction above and execute it consistently across all pages.**

### Typography Requirements

**DO NOT USE:**
- Arial, Times New Roman, Helvetica
- Inter, Roboto (overused in AI designs)
- System fonts

**RECOMMENDED FONT PAIRINGS (Google Fonts):**
- Playfair Display + Lato
- Cormorant Garamond + Montserrat
- Crimson Text + Work Sans
- DM Serif Display + Outfit
- Libre Baskerville + Source Sans Pro
- Lora + Nunito Sans

Choose fonts that match your selected aesthetic direction.

### Color Palette

Create a cohesive color scheme with:
- Primary color (brand color, used for CTAs and accents)
- Secondary color (supporting color)
- Neutral colors (backgrounds, text, borders)
- Accent color (highlights, special elements)

**Egyptian/Desert-Inspired Palettes to Consider:**
- Sand, terracotta, deep teal, cream
- Warm beige, olive green, burgundy, gold
- Dusty rose, sage green, cream, charcoal
- Ochre, deep blue, warm gray, ivory

Use CSS variables (custom properties) for easy color management:
```css
:root {
  --color-primary: #...;
  --color-secondary: #...;
  --color-accent: #...;
  --color-text: #...;
  --color-bg: #...;
}
```
